# Uplife Jobs Platform 

A modern job listing web application designed to connect candidates and recruiters through a secure and scalable backend architecture.

This was my first project using backend and API, and demonstrates full-stack development skills with focus on backend engineering, REST API design, authentication, and database management.

---

## Features

-  Company Profiles
-  Job Posting Management
-  Job Search & Filtering
-  RESTful API Architecture
-  Database Integration
-  Structured Backend Layering
-  Basic API Testing

---

##  Architecture

The application follows a layered architecture pattern:

Controller → Service → Repository → Database

This ensures:
- Clean separation of concerns
- Scalability
- Maintainability
- Easier testing
